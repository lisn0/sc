# Client Server Project

## Getting Started

### Dependencies

* Linux
* rfoo
* python3
* cython
* twitter
* requests

### Installing prerequisites

```shell
sudo apt-get install build-essential
pip install cython twitter requests
```

### Installing rfoo

```shell
git clone https://github.com/aaiyer/rfoo
cd rfoo
sudo python setup.py install
```
